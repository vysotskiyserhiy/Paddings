## Extend touches zone

Add padding to responsive elements 
```swift
UIView().padding = 10
UILabel().padding = 10
UIButton().padding = 20
```

## Installation

Paddings is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'Paddings'
```
